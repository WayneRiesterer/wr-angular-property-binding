Passing Single Object

The data in this section is passed as a single object from AppComponent to PersonComponent.

Name changes here are also reflected back in AppComponent.

The reason for this is again due to the object being passed by reference.

AppComponent State Values

Name: {{ personObject.name }}, Age: {{ personObject.age }}

Passing Primitive Values

The data in this section is passed as primitive values from AppComponent to PersonPrimitiveComponent.

Name changes in this case are not propagated back to AppComponent as they are passed by value and not by reference.

AppComponent State Values

Name: {{ name }}, Age: {{ age }}
